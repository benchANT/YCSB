### Required environment variables

- AZURE_TENANT
- AZURE_CLIENT_ID
- AZURE_SECRET

#### Options parameters

- dataexplorer.database < name string > :
  - Name of the database to use.
  - Default: ""

- dataexplorer.endpoint < name string > :
  - The URL of the database.
  - Default: ""
